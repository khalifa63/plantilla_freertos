Reglas de oro del flujo
0. Todo proyecto nuevo parte de proyecto_base.zip: nunca construir la infraestructura a mano.
1. Las tres primeras herramientas generan el prompt; el código siempre lo escribe Claude.
2. La respuesta de cualquiera de las tres se procesa con el mismo Ensamblador.
3. Tras agregar o borrar archivos del sketch: cerrar y reabrir Arduino IDE antes de compilar.
4. Si al subir aparece "No changed sectors found", el archivo nuevo no entró al build (ver regla 3).
5. Verificar siempre en el monitor serial (115200) la línea [MODULES] init '...' OK de cada módulo.
